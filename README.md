# 🏥 Healthcare Analytics Dashboard — Power BI

## 📌 Project Overview
An end to end Power BI dashboard analyzing healthcare data 
containing patient admissions, billing, diagnoses, and doctor 
performance across multiple hospitals.

## 📊 Dashboard Pages
- *Page 1 — Key Insights* 
- *Page 2 — Project Summary*
- *Page 3 — Overview* : KPI Cards, Monthly Trend, Map
- *Page 4 — Patient Analysis* : Age Groups, Admission Types, Test Results, Gender distribution
- *Page 5 — Doctor & Financial Analysis* * : Billing by Condition and Insurance & Top Doctors, Performance matrix

## 🎯 Key Insights
- Total patients analyzed : Total 1856 Patients analyzed across all hospitals.
- Most common condition :  Cardiovascular disease is the most common medical condition among patients
- Highest billing condition : Emergency Admission accounts for the highest percentage of total admissions
- Average length of stay : 16 days

## 🛠️ Tools and Techniques
- *Tool* : Microsoft Power BI Desktop
- *Data Cleaning* : Power Query Editor
- *Data Modeling* : Star Schema
- *DAX Measures* : 20 Custom Measures
- *Visuals* : 15 Plus Charts and Visuals
- *Interactivity* : 6 Slicers with Cross Page Filtering

## 📁 Dataset
- *Source* : Kaggle Healthcare Dataset
- *Records* : 1856 Patients
- *Columns* : 18 Fields
